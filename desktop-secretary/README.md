# 桌面小秘书 🐾

> 以**抽象治愈系桌面宠物**为载体的**智能待办事项管理工具**

## 功能特色

- 🐾 **桌面宠物**：3种可选角色（团子/云朵/小精灵），根据任务状态展示8种动画情绪
- ✅ **智能待办**：四象限快速创建，一键直达关联文件/文件夹
- 🔍 **智能匹配**：点击待办自动匹配并打开文件夹内的核心文件
- ⏰ **DDL管理**：截止时间提醒，逾期时宠物发出警告动画
- 📊 **智能排序**：基于使用频率、象限权重、时效的动态排序

## 使用说明

### 创建待办
1. 鼠标悬浮在宠物上（300ms），面板自动展开
2. 在底部输入框输入标题，支持 `!`（重要）、`@`（紧急）快捷符号
3. 按回车，在四象限选择器中选择所属象限

### 关联文件
- 直接从文件管理器**拖拽**文件/文件夹到待办卡片
- 或点击卡片上的 📎 / 📂 按钮选择文件/文件夹

### 一键打开
点击卡片的 ▶ 按钮，系统自动：
1. 打开所有关联文件夹
2. 基于标题关键词智能匹配并打开最相关的文件

### 宠物动画状态
| 状态 | 触发条件 |
|------|---------|
| 常态 | 无特殊事件 |
| 待办提醒 | 存在重要且紧急的待办 |
| 焦虑 | 待办总数 > 10 件 |
| 濒临逾期 | 任意 DDL 距今 ≤ 60分钟 |
| 完成庆祝 | 待办全部完成 |
| 交互反馈 | 点击/拖拽宠物 |

## 数据存储

所有数据存储在本地，**无任何网络请求**：
- 待办数据：`%AppData%/DesktopSecretary/todos.json`
- 应用设置：`%AppData%/DesktopSecretary/settings.json`
